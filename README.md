## Project Overview
This repository contains a collection of programming assignments completed during the CMPUT 175 course at the University of Alberta. These projects focus on applying fundamental computer science principles—such as custom abstract data types (ADTs), object-oriented design, standard file handling, data analysis and predictive model deployment in Python.

## Core Technologies & Libraries Used
* **Language:** Python 3 (Scripts & Jupyter Notebook)
* **Data Structures:** Dictionaries and Circular Doubly Linked Lists
* **Libraries:** `matplotlib` (for creating charts) and `sklearn` (for building the decision tree model)

---

### 🔹 Assignment 1: Global Trade Analysis
* Parses and extracts global trade data from multiple related text files.
* Uses a custom **`Table` class** to organize raw data obtained from the text files and manipulate it. 
* Determines the best locations to source products in order to minimize government tax tariffs and maximize company profit margins.
* All results are displayed in formatted tables in the terminal.

### 🔹 Assignment 2: Employee Attrition Risk Analyzer
* **Data Preprocessing:** Cleans a fictional dataset created by IBM data scientists by handling missing values.
* **Data Visualization:** Uses `matplotlib` to visually inspect the data.
* **Predictive Modeling:** Uses `sklearn` to train a simple Decision Tree model that calculates the risk score and predicts who is most likely to leave the company.
* **Review Workflow:** Implements a **Circular Doubly Linked List** (`Carousel`) to store employee records. This allows a user to interactively cycle forward and backward continuously through profiles, flag specific cases.
